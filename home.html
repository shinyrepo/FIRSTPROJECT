HTML basics

html is an acronym of hypertext markup language.

Hyper Text simply means "Text within Text".

A markup language is a programming language that is used to make text more interactive and dynamic.

HTML was created by Berners-Lee in late 1991 but "HTML 2.0" was the first standard HTML specification which was published in 1995
HTML 4.01 was a major version of HTML and it was published in late 1999.

HTML is being widely used to format web pages with the help of different tags available in HTML language.
